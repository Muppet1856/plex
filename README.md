# Plex DVR Postprocessing
Bash scripts for Plex Postprocessing

## Prerequisites
1. Comcut
2. Handbrake-CLI

## My Environment
* PMS (latest) running on a virtual Ubuntu 16.04
* 4vCPU / 2GB RAM
* AMD A-10 6800k APU in host running Windows 10
